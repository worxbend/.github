# worxbend

A personal open-source lab focused on Linux desktop tooling, systems programming, and developer utilities. Projects here tend to be small, direct, and built to scratch a real itch — not frameworks, not platforms, just things that work on a Linux desktop and stay out of the way.

## Active Projects

### 🖥️ Desktop & GNOME

| Project | Description | Stack |
|---|---|---|
| [airgradient-desktop](https://github.com/worxbend/airgradient-desktop) | GTK 4 + libadwaita dashboard for AirGradient air quality monitors | Rust, GTK4, libadwaita |
| [airgradient-gnome-extension](https://github.com/worxbend/airgradient-gnome-extension) | GNOME Shell panel extension for the same AirGradient local-server workflow | JavaScript |
| [scenedeck](https://github.com/worxbend/scenedeck) | Linux desktop controller for OBS Studio via WebSocket | Rust, GTK4, Tokio |

### 🛠️ Developer Utilities

| Project | Description | Stack |
|---|---|---|
| [dotbot-go](https://github.com/worxbend/dotbot-go) | Declarative dotfiles manager — symlinks, directories, shell commands | Go |
| [nerd-font-installer](https://github.com/worxbend/nerd-font-installer) | Install your favourite Nerd Fonts with one command | Go |
